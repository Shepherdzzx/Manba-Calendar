# 语音命令示例（v1）

本文给出第一版示例集，展示从用户语音/ASR 文本到结构化 JSON 的目标结果。

所有 JSON 都应符合 `docs/protocol.md`。

## 示例 1：新增明天下午的会议

**用户语音**：明天下午三点和张三开会  
**ASR 识别结果**：明天下午三点和张三开会

```json
{
  "intent": "create_event",
  "title": "和张三开会",
  "date": "2026-05-30",
  "time": "15:00",
  "need_reminder": true
}
```

## 示例覆盖说明

这组示例覆盖了第一版最重要的表达：

- 今天、明天、后天
- 上午、下午、晚上
- 几点、几点半
- 周几、下周几
- 新增、查询、删除三类 intent

后续如果增加“修改事件”“重复事件”“提醒偏移时间”等能力，应继续扩展示例集。
